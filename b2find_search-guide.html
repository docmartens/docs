B2FIND User Guide

The B2FIND discovery portal supports user-friendly navigation and powerful search functionalities. This includes

  • Free text search over the full text body
  • Geospatial and 'Timeline' search, i.e. search for all datasets, that cover a choosen region or time periode.
  • Other 'faceted' search, e.g. selecting values from 'facets', i.e. searchable metadata fields
These search requests can be combined and excecuted in one go. This page describes the usage of the B2FIND portal step by step and lists several tips and tricks that you may find useful to this effect.

Access B2FIND portal

You can access the B2FIND catalog at http://b2find.eudat.eu.

From the entry page you can list all communities, that provide metadata to B2FIND by clicking 'Communities'. By entering text in the provided free text fields 'Search your data' you can directly perform a full text search over the whole catalogue. If you want to use more subtle and particularized search functionalities, click the button 'Facetted Search'. It's usage is described in the following sections.

Geospatial Search

In the worldmap widget in the left upper corner you can select a region by drag and drop. This triggers a search for all datasets their spatial extention has an intersection with the selected region.

  1. Clear possibly previous search request by clicking the Clear button of the 'Filter by location' interface in the navigation bar.

  2. Selection of a region from the world map

    1. Click on the 'Draw a rectangle' button in the right upper corner of the worldmap wiget to start the region selection. Drag with the mouse over the wished rectangle, its borders will be marked red. Finally press the 'Apply' button to excecute the search request.

    2. Search result : If excecution of the query is finished, all datasets their spatial extention has an overlap with the selected region are listed in the right panel.

    Temporal Search

    B2FIND provides two kinds of search for date and time :

    • Selection of PublicationYear, i.e. the year the dataset is 'published' in.
    • 'Time line' search, i.e. search for all dataests, which cover a choosen time periode.

    Selection of PublicationYear

    In the search widget 'Publication Year' select first a start year by clicking in the left field. In the opened select window you can switch to the right decade (click on '<<' rsp. '>>') that the wished year comprises and where you can select it by clicking. The end year is similar selected in the right field.

    1. Clear possibly previous search request by clicking the Clear button of the 'Publication Year' interface in the navigation bar.

    2. Select a periode of publishing years

      1. Select a start year : Click on the left text field in the interface 'Publication Year'. This results in opening a select panel showing the years of the current decade. By clicking on the '<<' or the '>>' button you can switch to the previous or subfollowing decade. Finally select the wished start year by clicking on it in the decadal panel.

      2. Select a end year : Click on the right text field in the interface 'Publication Year'. This results in opening a select panel showing the years of the current decade. By clicking on the '<<' or the '>>' button you can switch to the previous or subfollowing decade. Finally select the wished end year by clicking on it in the decadal panel. This selection will submit the search request.

    3. Search result : If execution of the query is finished, all datasets that are published within the the choosen years are listed in the right panel.

    Selection of Temporal Coverage

    In the search widget 'Timeline' you can select a wished time periode, i.e. the temperol coverage searched for, by zooming in date and time histograms as described in the following.

    1. Clear possibly previous search request by clicking the 'Clear' button of the timline interface in the navigation bar.

    2. Clicking the button Timeline in the navigation panel opens the time line chart.

    3. Select a periode by dragging the mouse with pressed left in the 'histogram' at the bottom. This causes the openening the datasets/time diagram over the choosen periode the upper part of the chart.

    4. Zooming in time

      1. 'Drag and drop' with the left mouse button hold over a time interval in the upper datasets/time graph (zoomed part is shown). This results in the shrinking of the time line on the selected interval.

      2. (optional) : Reset the last zoom by clicking the button 'Reset zoom'.

      3. (optional) : Repeat zooming until the desired section is shown in the chart.

    5. Select a time interval by holding "Ctrl" (Win/Linux) or "Cmd" (Mac) down and clicking on two points (start and end time) in the chart line. This effects that start and end time are set in the datasets/time graph.

    6. Execution of the query is executed by closing the "Timeline" chat (clicking the "X"-button or the "Esc"-key or background). The chart disappears and search page is updated. I.e. selected 'Start' and 'End' timea are displayed (as integers) in the corresponding fields in the navigation bar and only datasets with corresponding temporal coverage are listed in the right side of the window.

    7. Further detailed and extented search can be requested by editing the Start and End fields in the navigatin bar's interface 'Timeline'
